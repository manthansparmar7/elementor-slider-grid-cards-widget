Create a versatile 'Cards Slider / Grid' widget for Elementor with advanced customization and seamless functionality.

== Description ==
Elementor Slider Grid Cards Widget provides users with a powerful Elementor addon to create a card slider or grid structure. Each card can include a heading, image upload, and other customizable fields.

The plugin offers a variety of Elementor widget settings to fully customize the design, giving you flexibility over how the cards are presented.

== Features ==
* Add cards in slider or grid structure.
* Customizable card layout with options for headings, images, and more.
* Advanced design customization options via Elementor widget settings.
* Seamless integration with Elementor's drag-and-drop interface.

== Purpose ==
The plugin provides an easy and intuitive way for Elementor users to add customizable cards in a slider or grid format. Users can personalize their cards with headings, images, and other content, while enjoying complete design control using Elementor widget settings.

== Installation ==
1. Download the plugin and upload the ZIP file via the WordPress dashboard or manually place it in the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The widget will be available in Elementor under the Widgets section for use in your pages or posts.

== Usage ==
Once activated, the 'Cards Slider / Grid' widget will appear in the Elementor panel. Simply drag and drop the widget to your desired page or post and begin customizing the cards with your preferred settings.
